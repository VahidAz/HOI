# HOI

#### Configs
* Ubuntu 18.04.3
* python 3.7.5
* gcc 7.4.0
* NVIDIA Driver 396.54
* CUDA 9.1
* conda create -n HOI python=3.7.5
* conda install pytorch=0.4.1 cuda91 -c pytorch
* conda install torchvision
* Make packages in models folder(sh make.sh)
* mkdir checkpoints
* mkdir datasets
* mkdir pretrained_models
